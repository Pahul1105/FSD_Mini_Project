# Java Backend Notes

This folder contains a beginner MVC backend using:

- JSP for simple view
- Servlet for controller
- DAO classes for database work
- MySQL for data storage

Main API paths:

- `GET /jobportal/api/jobs`
- `POST /jobportal/api/jobs/add`
- `POST /jobportal/api/jobs/delete?id=1`
- `POST /jobportal/api/apply`

Place the MySQL JDBC jar and Servlet container libraries in your Tomcat/Eclipse setup before running.
